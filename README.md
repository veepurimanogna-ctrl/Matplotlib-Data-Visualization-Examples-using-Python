# Matplotlib-Data-Visualization-Examples-using-Python
This repository contains a collection of beginner-friendly Python programs that demonstrating various data visualization techniques, including bar charts, line graphs, scatter plots, and pie charts. This project is designed for beginners to learn the fundamentals of data visualization 


# Python Matplotlib Visualization Examples

## Overview

This repository contains a collection of Python programs that demonstrate the use of the **Matplotlib** library for creating different types of data visualizations. The examples are simple, easy to understand, and designed for beginners who want to learn graphical representation of data using Python.

## Features

- Bar Charts
- Line Graphs
- Scatter Plots
- Pie Charts
- Different Colors and Styles
- Various Marker Types
- Simple and Well-Structured Code

## Technologies Used

- Python 3
- Matplotlib

## Programs Included

### 1. Student Count Bar Chart
Displays the number of students enrolled in different programming language courses using a bar chart.

### 2. Monthly Sales Line Chart
Plots monthly sales data using line graphs with different colors and line styles.

### 3. Study Hours vs Marks Scatter Plot
Visualizes the relationship between study hours and student marks.

### 4. Scatter Plot with Different Markers
Demonstrates scatter plots using square (`s`), star (`*`), and triangle (`^`) markers.

### 5. Programming Language Distribution Pie Chart
Represents the percentage distribution of students across different programming languages.

### 6. Employee Department Bar Chart
Shows the number of employees in different departments using a bar chart.

### 7. Monthly Temperature Line Chart
Displays temperature changes over six months using a line graph.

### 8. Age vs Salary Scatter Plot
Illustrates the relationship between employee age and salary.

### 9. Product Sales Bar Chart
Shows sales performance of different products using a bar chart.

### 10. Test Marks Line Chart
Displays marks obtained in multiple tests using a line graph.

### 11. Study Hours vs Marks Scatter Plot
Another example showing how study hours affect student performance.

## Learning Objectives

After completing these examples, you will learn:

- How to import and use Matplotlib.
- How to create different chart types.
- How to customize graphs with colors, markers, and line styles.
- How to add titles and axis labels.
- Basic techniques for visualizing datasets in Python.
